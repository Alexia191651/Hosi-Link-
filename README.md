# HOSI-LINK
IS Project 1 - Hospital Link System

A centralized healthcare management system designed to streamline coordination between paramedics, ICU bed management staff, and blood bank personnel.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Tech Stack
- Next.js (JavaScript)
- Maria DB
- NextAuth.js
- React Context + SWR
